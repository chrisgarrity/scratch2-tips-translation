Tips Window Images

This folder contains images used in the tips window. You can download the full collection as a zip file (36.7Mb).

Here are the steps for translating the tips images:

  1. Make your own version of each image that needs translating (only some images have text)
    • Make a screenshot in Scratch that is equivalent in your language.
    • Use an image editor program to make the image size the same, if possible - but no wider than 291 pixels.
    • If it's an animated gif, you can either provide a static image instead, or make an animated gif. To make an animated gif:
      1. Make a screen recording of dragging the blocks (for example, using Camstudio on Windows or Quicktime on Mac).
      2. Import that movie into a GIF creating program (such as Photoshop, GIMP, Gifcam, or GIFBrewery).
  2. Save your translated images in a new folder keeping the same file name, but altering the extension if you changed the image format (for example, if you changed a gif to jpg). The extension can be png, jpg or gif.
  3. Send translated images to translate@scratch.mit.edu. Or add them to a shared folder like dropbox or google drive, and send translate@scratch.mit.edu a link to the folder.

Notes:

  • Images with names that start with a number are for the Getting Started tutorial.
  • ayn-...: Animate Your Name tutorial
  • ca-...: Catch Game tutorial
  • da-...: Let's Dance tutorial
  • fa-...: Favorite Things tutorial
  • fl-...: Make It Fly tutorial
  • hi-...: Hide and Seek tutorial
  • int-...: help related to the Scratch Editor interface
  • mu-...: Make Music tutorial
    • pg-...: Pong Game tutorial
  • ra-...: Race Game tutorial
  • vp-...: Virtual Pet tutorial
  • Names that start with bb-, bs-, hoc-, od-, va- are not needed, they are for obsolete tutorials.
